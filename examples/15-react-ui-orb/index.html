Deepgram Orb React UI

The signature Deepgram hoop animation — state-driven, no audio source required. Click the states below to see the orb deflate, inflate, speed up, and rock.

State-driven (default)

import { Orb } from '@deepgram/agent-react-ui';

// Just pass the state — no audio source needed
<Orb state={"listening"} />

// Custom colors
<Orb state={"speaking"} colors={["#6366f1", "#ec4899"]} />

// Optional audio reactivity
<Orb
  state={"speaking"}
  getOutputVolume={() => player.getOutputVolume()}
/>
