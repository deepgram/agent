Deepgram Orb React UI

The signature Deepgram hoop animation — 4 gradient crescent arcs that rotate, pulse, deflate, and react to audio volume. Ported from deepgram/browser-agent.

Code

import { Orb } from '@deepgram/agent-react-ui';

<Orb
  state={mode}
  agentVolume={getOutputVolume()}
  userVolume={getInputVolume()}
  size={200}
/>
